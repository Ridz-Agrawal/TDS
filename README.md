Matrix Build Demo (f338154)

This repository demonstrates a GitHub Actions matrix build strategy with artifact management, as required by task f338154.

The workflow is located in .github/workflows/matrix-build.yml.

It runs a matrix build for three different Python versions ('3.9', '3.10', '3.11').

Each job generates and uploads a unique build artifact.
